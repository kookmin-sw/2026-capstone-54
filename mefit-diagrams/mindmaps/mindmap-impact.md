## 목적

본 캡스톤이 만들 수 있는 **5 가지 기대 효과** 를 영역별로 시각화합니다. 사용자 / 기술 / 사회 / 학습 / 사업 측면의 가치를 표현합니다.

## 표현 내용

5 영역 (대략):
1. **사용자 가치** — 자기주도 면접 트레이너 + 즉각 피드백 + 학습 루프
2. **기술 가치** — LLM 응용 아키텍처 + 멀티모달 (영상 / 음성) 분석 + 마이크로서비스
3. **사회 가치** — 면접 사교육 격차 완화 + 비대면 학습 + 데이터 기반 채용
4. **학습 / 교육** — 학습 데이터 자산화 (멘토링 인사이트) + 패턴 분석
5. **사업 / BM** — Free / Pro 구독 + 티켓 + 데이터 자산화 BM
